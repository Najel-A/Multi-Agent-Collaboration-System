4 · Multi-Agent Layer (15 Agents)
Orchestrate 15 specialized agents across 4 progressive coordination models. M1 (6 agents): Ingestion, Preprocessing, Anomaly Detection feed Planner (Mistral-7B), Executor, Reviewer. M2 (+1): Retrieval Agent with FAISS/RAG. M3 (+2): Intent Classifier + Validation Agent on shared Blackboard. M4 (+5): 3 Debaters (Opus/GPT-4o/DeepSeek-R1), Referee, Safety Agent (HiTL), Context Memory. Tier 1 = autonomous; Tier 2-3 = human approval required.

5 · Model Architectures (Coordination Patterns)
Four progressively sophisticated agent coordination patterns. Each model inherits agents from the previous, adding a new coordination layer. Ranges from a linear pipeline (M1) to a self-evolving debate system with safety and memory (M4).
M1 Baseline Pipeline
M2 RAG + Shared Knowledge
M3 Blackboard + Validation
M4 Debate + Safety + Memory

Model 1 — Centralized Linear Pipeline (Baseline) — 6 agents

Sequential message-passing pipeline. Classical anomaly detectors (Isolation Forest, One-Class SVM, LOF) flag outliers. Fine-tuned Mistral-7B LoRA classifies root cause into 12 families. ~$0/incident.

Data Ingestion --> Preprocessing --> Anomaly Detection --> Planner --> Executor --> Reviewer [message] [message] (IF, SVM, LOF) (Mistral-7B [message] [message] [message] LoRA) [message]

Model 2 — RAG-Enhanced Pipeline — 7 agents (+1)

Adds a Retrieval Agent and FAISS Knowledge Base. Planner upgraded to Qwen2.5-7B LoRA with RAG context for full structured remediation. ~$0/incident.

M1 Pipeline --> [NEW] Retrieval Agent --> [Knowledge Base (FAISS)] | Planner (Qwen2.5-7B LoRA, RAG-enhanced) --> Executor --> Reviewer

Model 3 — SMART Blackboard Architecture — 9 agents (+2)

Centralized Blackboard shared state. Adds Intent Classifier (GPT-4o-mini/Haiku) and Validation Agent (hallucination detector). Planner upgraded to Claude Opus 4 / GPT-4o. ~$0.08-0.15/incident.

+------- BLACKBOARD (shared state) -------+ | intent / retrieved_context / | | validation_result / rca_plan | +--+------+----------+----------+-----------+ | | | | [NEW] Intent Retrieval [NEW] Planner --> Executor Classifier Agent Validation (Opus/GPT-4o) (GPT-mini) (M2) Agent (GPT-4o/Sonnet)

Model 4 — Self-Evolving Debate MAS — 14 agents (+5)

Three Debater Agents (Opus, GPT-4o, DeepSeek-R1) generate competing hypotheses. Referee scores by evidence (35%), coherence (25%), specificity (20%), resilience (20%). Safety Agent (HiTL) gates high-risk actions. Context Memory enables long-term self-improvement. ~$0.30-0.60/incident.

7 · Observability & Outputs
Collect, index, visualize, and act on incident data. Filebeat ships container logs to Logstash for parsing/normalization. Elasticsearch indexes into daily indices. Kibana provides interactive dashboards. Targets: F1 > 0.85, AUC-ROC > 0.90, MTTD < 5m, MTTR < 30m.
